# Electrical Vertical Guide

Industry-specific guidance for generating electrical contractor websites.

## Industry Overview

Electrical contractors provide installation, repair, and maintenance of electrical systems. Key characteristics:

- **Safety-critical**: Electrical work has serious safety implications
- **Code-driven**: Must comply with NEC and local codes
- **Growing demand**: EV chargers, smart homes, solar
- **High trust required**: Dangerous if done improperly

## Services to Generate

### Primary Services (always include)

1. **Panel Upgrades**
   - 100A to 200A+ upgrades
   - Features: increased capacity, safety, code compliance

2. **Whole Home Rewiring**
   - Complete electrical system replacement
   - Features: aluminum wire replacement, older home updates

3. **Outlet & Switch Installation**
   - New outlets, GFCI, USB outlets, switches
   - Features: all locations, code compliant, smart switches

4. **Lighting Installation**
   - Indoor, outdoor, recessed, landscape
   - Features: LED upgrades, dimmer installation, design

5. **Generator Installation**
   - Standby and portable generator hookup
   - Features: automatic transfer switch, sizing, permits

### Secondary Services (include 2-3)

6. **EV Charger Installation**
   - Home electric vehicle charging stations
   - Features: Level 2 chargers, Tesla, universal

7. **Electrical Troubleshooting** (emergency: true)
   - Diagnosis of electrical issues
   - Features: no power, circuit tripping, burning smell

8. **Electrical Safety Inspection**
   - Home electrical assessment
   - Features: code compliance, insurance, real estate

## Licenses & Certifications

### State Licenses (vary by state)
- **Arizona**: ROC license (electrical specialty)
- **California**: CSLB C-10 (Electrical)
- **Texas**: TDLR electrical license
- **Florida**: State certified/registered electrical contractor

### Industry Certifications
- Master Electrician (state-issued)
- Journeyman Electrician
- Tesla Certified Installer
- Generac Authorized Service Dealer
- BBB A+ Rating
